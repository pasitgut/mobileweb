Lab 4: Photo Gallery ด้วย Ionic Vue

รายวิชา Mobile Web Application Development

ผู้จัดทำ: พสิษฐ์ ผลวิเศษพรสุข (663380020-7)

รายละเอียดงาน

แอปพลิเคชัน Photo Gallery พัฒนาด้วย Ionic Vue โดยมีความสามารถดังนี้:

  • ถ่ายภาพโดยใช้ Capacitor Camera Plugin
  • บันทึกไฟล์ลงในเครื่องด้วย Capacitor Filesystem
  • แสดงผลรูปภาพในรูปแบบ Grid
  • โหลดข้อมูลรูปภาพที่บันทึกไว้กลับมาแสดงผลได้
ดาวน์โหลดแอปพลิเคชัน (.apk)

สามารถดาวน์โหลดไฟล์เพื่อติดตั้งบน Android ได้ที่นี่
